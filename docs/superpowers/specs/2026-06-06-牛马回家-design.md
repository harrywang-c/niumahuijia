# 牛马回家 — 游戏设计文档

**日期：** 2026-06-06  
**类型：** 横版卷轴解谜平台游戏  
**技术栈：** Phaser 3（本地离线）+ Matter.js 物理引擎  
**交付形态：** 多文件结构（`index.html` + `js/phaser.min.js` + 资源目录）  
**目标：** 比赛提交，三关完整体验

---

## 世界观

马头小人下班后迷失在奇异世界，需要用随身携带的传送门枪和神奇墨水，躲开巡逻牛仔，回到自己温暖的家。

---

## 核心机制

### 1. 角色操控
- `←` `→`：左右移动
- `Space`：跳跃（单段跳）
- **鼠标左键拖拽**：画墨水
- **鼠标右键单击**：射出 Portal（第一次=蓝门，第二次=橙门，第三次=覆盖蓝门）

### 2. 墨迹系统
- 玩家在屏幕上拖拽鼠标，实时绘制墨迹笔画
- 松开鼠标后，笔画固化为 **Matter.js 刚体多边形**（按笔画路径生成碰撞体）
- 固化后的墨水平台受重力影响，会自然下落直到碰到固定地面
- **墨量限制**：每关有固定墨量条（UI 左上角显示），用完无法继续绘制
  - 关卡1：400px 笔画长度
  - 关卡2：280px
  - 关卡3：250px
- 墨水平台可作为 Portal 开门面（玩家可在墨水表面射 Portal）

### 3. Portal 传送门系统
- 每次右键射出光束，命中可开门面则创建传送门
- 同时存在蓝、橙两扇门；第三次射击覆盖最早的门
- 玩家/墨水物体进入一扇门，从另一扇门以**相同速度+方向修正**射出（保持动量）
- **可开门面**：固定墙壁（白色标记）+ 玩家绘制的墨水平台
- Portal 在关卡重置时消失

### 4. 牛仔敌人
- 在设定路径上左右巡逻
- 前方有**扇形视野锥**（角度60°，距离可配置）
- 玩家进入视野锥 → 触发抓捕动画 → 关卡完全重置（墨水、Portal、位置全清）
- 墨水可以遮挡视野锥（墨迹当掩体）

### 5. 危险区域
- **尖刺**：接触即重置关卡
- **坑洞**：掉出屏幕底部即重置关卡
- **岩浆**（第三关）：接触即重置关卡

### 6. 胜利条件
- 马头小人到达关卡最右侧的「家」门口触发胜利动画，进入下一关

## 关卡设计

### 第一关 — 「墨与门的初见」

**难度：** ★★☆☆☆（新手引导关）  
**新手引导：** 通过屏幕上的提示箭头和文字框，逐步引导玩家学习每个机制

**地图布局（横向，约3屏宽）：**

```
[起点] ---坑1(窄)--- [小平台] ---坑2(宽)--- [高墙] ---平地--- [牛仔守门] [家]
                                              (Portal墙)          (背对玩家视野)
```

**教学序列：**
1. 起点右侧第一个小坑：提示文字「按住鼠标左键画墨水！」→ 引导画桥过坑
2. 第二个宽坑：墨量明显不够直接填满，提示「墨水不够用了！试试传送门吧」
3. 高墙：Portal 标记面在墙左右两侧，提示「右键点击射出传送门」
4. 通过高墙后：一个静止牛仔背对玩家（不会看到），但家门口视野指向入口
5. 需要在牛仔旁边墙上开 Portal 绕进家

**关卡特殊元素：**
- 墨量：400px（相对宽松）
- 无巡逻牛仔（仅一个静止守门）
- Portal 墙有明显视觉标记（白色发光边框）
- 提示箭头在前两个谜题完成后消失

---

### 第二关 — 「墨水不只是平台」

**难度：** ★★★☆☆  
**核心新机制：** 墨水可以遮挡牛仔视线

**地图布局（约4屏宽，分上中下三条路）：**

```
[上路] 高台 ============================== Portal传送点
[中路] [起点] ---Portal墙--- [固定牛仔守中路]--- 通道 ---[门]
[下路] 岩浆 ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
```

**谜题逻辑：**
- 中路有固定牛仔视野覆盖全中路，无法直接过
- 上路高台是安全路线，但需要先通过中路开 Portal 才能上去
- **解法：** 在牛仔视野路径上画竖向墨水遮挡视线 → 快速冲到 Portal 墙开门 → 传送到上路高台 → 高台走到终点上方 → 再开 Portal 落到终点
- 中间有一个来回巡逻的牛仔，需要时机配合
- 墨量：280px（仅够画一道掩体墙 + 少量辅助平台）

**陷阱设计：**
- 墨水画错位置自己也被挡住视线（无法看到敌人）
- 高台边缘有尖刺，Portal 落点需要精确

---

### 第三关 — 「最后一里路」

**难度：** ★★★★☆  
**三段式，每段独立谜题，依次解锁**

**第一段「坠落传送」：**
- 高台在左上，目标平台在右下深坑（有岩浆，仅中间有安全着陆点）
- 直接路线被岩浆封死
- 解法：右壁开 Portal B，高台左墙开 Portal A → 跳入 A 从 B 高速飞出落到安全点
- 需要估算抛物线角度，失误落入岩浆重置

**第二段「三牛夹击 + 压力板」：**
- 三个牛仔，视野几乎覆盖全区域
- 区域中间有一个压力板，踩下开门进入第三段
- 解法：在两个牛仔之间画墨水分割视野 → Portal 快速传送到压力板踩下 → 立刻 Portal 回来进门
- 时序严格：压力板踩下到门关闭有3秒窗口

**第三段「锁门与钥匙」：**
- 家就在眼前，但门上了锁
- 钥匙在房顶（有尖刺边缘无法直接跳上）
- 门口牛仔视野朝右，从右边必死
- 解法：天花板开 Portal B，左墙开 Portal A → 画墨水垫高 → 跳入 A 从天花板 B 飞出落房顶 → 取钥匙 → 从牛仔左侧盲区落地进门

## 游戏界面

### 进入界面（Title Screen）
- 全屏背景：夜空 + 远处温暖小屋剪影
- 标题文字：「牛马回家」（像素字体，带墨迹滴落效果）
- 副标题：「用墨水和传送门找到回家的路」
- 按钮：「开始游戏」
- 背景有马头小人慢慢走动的动画

### HUD（游戏内）
- 左上角：墨量条（墨水笔形状图标 + 进度条）
- 右上角：关卡编号
- Portal 状态：屏幕底部显示蓝/橙门当前状态（已开/未开）

### 关卡重置
- 屏幕红色闪烁
- 文字提示：「被抓了！」或「掉坑了！」
- 0.8秒后淡入重置

### 胜利界面
- 烟花/墨点爆炸效果
- 文字：「回家了！」
- 进入下一关按钮 / 最终关显示「通关！」

## 技术架构

### 文件结构
```
牛马回家/
├── index.html          # 游戏入口
├── js/
│   ├── phaser.min.js   # Phaser 3 离线包
│   └── matter.min.js   # 可选，Phaser内置
├── src/
│   ├── main.js         # 游戏初始化
│   ├── scenes/
│   │   ├── TitleScene.js
│   │   ├── Level1Scene.js
│   │   ├── Level2Scene.js
│   │   └── Level3Scene.js
│   └── systems/
│       ├── InkSystem.js    # 墨水绘制+物理
│       ├── PortalSystem.js # 传送门逻辑
│       └── GuardSystem.js  # 牛仔视野+AI
└── assets/
    ├── sprites/        # 角色+牛仔像素图
    └── audio/          # 音效（可选）
```

### 关键技术点
- **墨水固化**：收集鼠标路径点 → 用 `Phaser.GameObjects.Graphics` 渲染 → Matter.js `Bodies.fromVertices()` 创建物理体
- **Portal 传送**：检测玩家与 Portal 碰撞 → 计算出口位置和速度向量 → `matter.Body.setPosition()` + `setVelocity()`
- **视野锥体**：以牛仔位置为顶点射出多条射线 → 检测是否命中玩家且中间无障碍物（包括墨水）

## 美术风格

- **基调：** 像素风（16-bit），色彩明亮但有层次
- **马头小人：** 棕色马头 + 白色身体，携带小型portal枪，32x32精灵
- **牛仔：** 戴牛仔帽，蓝色/棕色服装，视野锥为黄色半透明扇形
- **墨水：** 纯黑色，边缘略带墨晕效果
- **Portal：** 蓝色/橙色发光椭圆，带粒子效果
- **环境：** 关卡1白天草地，关卡2黄昏城镇，关卡3黑夜工厂
