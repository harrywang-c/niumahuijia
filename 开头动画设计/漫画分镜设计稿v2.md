# 《牛马回家》开场动画 · 分镜漫画设计稿 v2

**风格参考**：LANND HOOK / 打工人be like 系列
**版式**：16:9竖屏 · 每页3格等高 · 共4页12格
**播放方式**：逐格动态漫画，连续叙事
**主色调**：明黄 #FFCC00 · 薄荷蓝 #A8D8D8 · 纯白 #FFFFFF

## 页面排版规则

```
┌─────────────────┐
│     第1格        │  ← 占页面1/3高度
├─────────────────┤
│     第2格        │  ← 占页面1/3高度
├─────────────────┤
│     第3格        │  ← 占页面1/3高度
└─────────────────┘
```

格与格之间留6px黑色分隔线。
文字/气泡可溢出格子边界，压在分隔线上，增加漫画动感。

## 第1页（第1-3格）｜周五19:00，下班了！

**整页背景色**：明黄 #FFCC00

### 第1格｜城市夜景定场

**构图**：
- 背景：夜晚城市剪影，深蓝/黑色办公楼群，大量黄色窗格亮灯
- 前景左下：马头人侧脸剪影趴在工位，眼神空洞望向窗外
- 右上角黄色圆角标签：`SKETCH DAILY LIFE 2026`

**画面文字**（大号黑色手写体，左上）：
> 这是一个
> 普通的周五……

**溢出**："周五"二字轻微压入第2格上边线

---

### 第2格｜时钟特写

**构图**：
- 画面中央：挂钟大特写，指针指向19:00，手绘风钟面
- 四周：日式放射速度线
- 左侧竖排音效字：`当 ——`（大号、有压迫感）
- 右下角：马头人只露头顶+两只竖耳，从格子下沿探入

**画面文字**（钟面下方）：`19:00`

**溢出**：马头人耳朵从下边线溢出进入第3格

---

### 第3格｜马头人猛然起身

**构图**：
- 马头人从工位猛地站起，椅子向后飞出
- 深灰西装，棕色公文包高举
- 背景同事轮廓剪影仍低头敲键盘（对比）
- 右侧椭圆气泡（最大字）：`下班！！！`（感叹号溢出气泡）
- 左下旁白小框：`已连续工作11小时`

**溢出**：公文包和感叹号压过上边线

---

## 第2页（第4-6格）｜障碍接连出现

**整页背景色**：明黄 #FFCC00

### 第4格｜迈出第一步

**构图**：
- 低角度仰拍：马头人大步迈出，腿部+皮鞋占画面下2/3
- 走廊地板透视线延伸向远处
- 远处出口：圆形暖光
- 马头人上半身从格子上边线溢出

**画面文字**（右侧竖排手写）：
> 回家
> 只需要
> 走出这扇门——

---

### 第5格｜消息轰炸

**构图**：
- 马头人站画面正中，被消息气泡从四面射来
- 表情：眼睛瞪成正圆，嘴巴大张，全身速度线
- 各气泡内容散落四周（手绘椭圆）：
  - `老板：晚点走～`（左上，最大）
  - `今晚全员加班！`（右上，红色边框）
  - `19:30 会议`（右侧）
  - `[群消息] 99+`（左侧）

**音效字**（右上角溢出格子）：`叮叮叮——！`

---

### 第6格｜老板大脸突袭

**构图**：
- 老板夸张大脸从上方黑暗俯视（只有脸，无身体）
- 眼神意味深长，嘴角微微上扬
- 马头人在下方仰头，身体僵直，腿部震动线
- 背景：恐怖感斜线阴影，格子整体偏暗
- 老板云朵气泡：`小明，今晚的需求……`
- 马头人锯齿OS框（右侧）：`不要说出来`

**溢出**：老板头顶从上边线溢出压入第5格

---

## 第3页（第7-9格）｜崩溃与泪水

**整页背景色**：薄荷蓝 #A8D8D8（情绪转折，与前两页形成节奏变化）

### 第7格｜彻底崩溃

**构图**：
- 马头人瘫坐地上，两腿前伸，公文包丢在旁边
- 全身密集黑色崩溃小烟雾符号
- 头顶浮着3-4个小红点（未读消息）

**画面文字**（上方，字体偏小，体现疲惫）：
> 这条路
> 怎么走都走不出去……

---

### 第8格｜泪水特写

**构图**：
- 马头人眼睛极度特写，占满整格
- 白色圆眼球，棕色眼眶，泪水在眼眶积聚
- 一颗晶莹泪珠悬在下睫毛边缘
- 瞳孔内：极小线条勾出"家"的模糊轮廓
- 格子四周留黑边（聚焦感）

**文字**：无（整格留白）

**溢出**：泪珠从下边线溢出，化作第9格的起点

---

### 第9格｜泪落成墨·传送门诞生

**构图**：
- 上半格：地面视角，泪珠落下轨迹线，落地晕染成深蓝水墨扩散（手绘有机形状）
- 下半格：第二滴泪落地后墨迹聚拢，形成蓝色发光传送门，边缘墨迹纹理
- 马头人右侧半身入镜，表情从崩溃变为好奇
- 音效字（左侧）：`滴——`
- 旁白框（右下圆角小框）：`泪水，是有分量的`

---

## 第4页（第10-12格）｜规则揭晓·出发回家

**整页背景色**：纯白 #FFFFFF（最终章，干净感）

### 第10格｜两个传送门

**构图**：
- 地面两个蓝色发光传送门，A在左，B在右偏远
- 两者之间蓝色虚线连接
- 马头人站A旁边，单手指向它，表情问号脸

**旁白框**（顶部居中）：
> 墨迹连成了路
> 泪水开了门

---

### 第11格｜玩法示意图

**构图**：
- 整格做成"游戏教学图"风格，像手绘说明书
- 左区：马头人手指划线，蓝色线条跟随延伸
- 中区：蓝线连传送门A→B，流动箭头
- 右区：家的手绘剪影，窗户透出黄色暖光
- 三区之间手绘箭头 `→` 连接
- 整格加手绘虚线外框

**画面文字**（顶部手写）：
> 画线 → 引导泪水 → 穿越传送门 → 回家

---

### 第12格｜出发

**构图**：
- 马头人背影居中，面朝画面深处
- 远方：家的剪影，橙黄暖光是整页最亮区域
- 脚边：两个传送门微蓝光
- 夜空：手绘 ✦ 形星星散落

**旁白框**（右下，手写体最小号）：
> 回家的路
> 要靠自己铺。

**底部大字**（全幅居中，最大号，最终停留帧，溢出页面底边）：
> 【开始回家】

## 制作规范

| 项目 | 规格 |
|------|------|
| 画面尺寸 | 1080×1920px（16:9竖屏） |
| 单格尺寸 | 1080×640px |
| 格间分隔线 | 黑色实线 6px |
| 字体 | 站酷酷黑 / 方正手迹体 |
| 对话气泡 | 手绘椭圆，黑线，白色填充 |
| 旁白框 | 圆角矩形，细黑线 |
| 溢出规则 | 重要元素（气泡/音效字/角色部位）可越过格线 |
| 每格停留时长 | 1.5秒，第8格停2.5秒 |
| 进场动效 | 从下往上滑入，0.3秒 |
| 配乐节点 | 第2格：钟声；第5格：密集叮声；第9格：水滴声；第12格：温暖弦乐 |
